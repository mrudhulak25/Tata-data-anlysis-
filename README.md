#  Tata GenAI Powered Data Analytics Job Simulation

##  Project Overview

This project was completed as part of the Tata GenAI Powered Data Analytics Job Simulation on Forage. The objective was to analyze customer credit risk data, build a predictive delinquency framework, generate business insights, and design an AI-powered collections strategy for a financial services company (Geldium).

##  Project Objectives

* Perform Exploratory Data Analysis (EDA) on customer financial data.
* Identify key risk indicators associated with credit delinquency.
* Develop a predictive modeling strategy for delinquency prediction.
* Translate analytical findings into business recommendations.
* Design an AI-driven collections strategy using responsible AI principles.

##  Project Tasks

### Task 1: Exploratory Data Analysis

* Analyzed customer financial behavior and risk factors.
* Identified missing values and data quality issues.
* Investigated relationships between credit utilization, missed payments, income, and delinquency risk.
* Generated insights using GenAI-assisted analysis.

### Task 2: Predictive Model Planning

* Designed a customer delinquency prediction framework.
* Selected Logistic Regression as the primary model.
* Defined evaluation metrics:

  * Accuracy
  * Precision
  * Recall
  * F1 Score
  * AUC-ROC
* Incorporated fairness and bias evaluation considerations.

### Task 3: Business Summary Report

* Converted analytical findings into business recommendations.
* Identified high-risk customer segments.
* Proposed proactive intervention strategies.
* Applied SMART recommendation principles.

### Task 4: AI-Powered Collections Strategy

* Designed an Agentic AI collections workflow.
* Defined automated and human-in-the-loop decision processes.
* Established Responsible AI guardrails:

  * Fairness
  * Explainability
  * Transparency
  * Privacy Protection
  * Regulatory Compliance

##  Key Insights

* High credit utilization is strongly associated with delinquency risk.
* Customers with multiple missed payments have a significantly higher probability of default.
* Younger customers with financial strain require proactive engagement.
* AI-driven interventions can improve recovery rates while maintaining customer trust.
